## 💻 项目经历

### 1. [邻购团 · 社区拼团交易平台](https://github.com/chenwenjun/project-neighborhood)

**技术栈**: SpringBoot, Mybatis, MySQL, Redis, RabbitMQ
**项目描述**: 设计并实现了一套拼团社交电商系统，涵盖从“优惠试算—锁单抢占—支付结算—退单”的全链路功能。
**核心职责**:

* 设计并实现拼团规则过滤引擎，采用责任链模式。
* 实现优惠折扣试算，通过多线程异步加载提升系统响应速度。
* 通过 Redis 位图实现用户精准筛选，并解决高并发性能瓶颈。

### 2. [智能业务助手（美食推荐 / 机票退费查询）](https://github.com/chenwenjun/project-intelligent-assistant)

**技术栈**: Dify Chatflow, Multi-Agent, RAG, vLLM, Qwen3
**项目描述**: 设计并实现多 Agent 架构，针对美食推荐与机票退费查询提供可扩展的解决方案。
**核心职责**:

* 设计多 Agent 系统架构，分类器 Router 用于意图分流。
* 结合结构化知识库与 RAG，实现可信的美食推荐与机票退费查询。
* 使用 Qwen3 保障数据安全与多轮对话的跨 Agent 上下文共享。

### 3. [汉服租赁秒杀与智能客服平台](https://github.com/chenwenjun/project-hanfuren)

**技术栈**: SpringBoot, Mybatis, Redis, RAG, Tool Calling, SSE
**项目描述**: 面向汉服租赁平台设计高并发秒杀系统，同时引入智能客服功能，通过 RAG 存储汉服知识库，提升用户体验。
**核心职责**:

* 设计秒杀活动与库存一致性方案，确保“一人一单”和防超卖。
* 实现多用户对话记忆隔离，支持 20 轮上下文记忆。
* 使用 RAG 知识库提供专业领域问答，减少幻觉并提升问答准确率。
